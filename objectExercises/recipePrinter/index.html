Object Exercise 2: Key Recipe

You want to build a recipe storage system that can print itself. As a prototype, you'll develop an object that holds information on your favorite recipe and displays all the information when the user asks for it.

  1. Create an object that has key-value pairs for the recipe's title (a string), number of servings (a number), and ingredients (an array of strings).
  2. As a last property/pair, include a method called printRecipe that logs the recipe information on separate lines when the user invokes it. The method should utilize the this keyword and use a loop for iterating over the ingredients. So, for example, the output should look like:
    • Title: Mandazi
    • Serves: 6
    • Ingredients:
    • egg
    • sugar
    • milk
    • butter
    • flour
    • baking powder
