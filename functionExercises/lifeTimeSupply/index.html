Function Exercise 3: Lifetime Supply Calculator

Ever wondered how much a "lifetime supply" of your favorite snack is? Wonder no more!

  1. Write a function named calculateSupply that takes 2 arguments: current age, and amount per day. It should calculate the amount the user would consume for the rest of his/her life (based on a constant max age, let's say 80, and 365 days per year). Last, it should output the result to the console. Here's an example output at 2 mandazi per day with a current age of 26 and max of 80: You will need 39420 mandazi to last you until the ripe old age of 80.
  2. Bonus: Accept floating point values for amount per day, like 1.5 pizzas, and use the global Math object to round the result to a whole number.
