Function Exercise 1: Maximum

Time to play with some numbers! Think you can calculate if one number is larger than another using Javascript? Let's find out!

  1. Define a function max() that takes two numbers as arguments and returns the largest of them. Use the if-then-else construct available in Javascript. If the numbers are equal, just return one of them.
  2. Define a function maxOfThree() that takes three numbers as arguments and returns the largest of them. Again, if there is a tie, just return one of the largest.
